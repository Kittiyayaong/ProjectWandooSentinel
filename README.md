# ProjectWandooSentinel

## Hands-on Lab Introduction
본 실습 과정은 한국 마이크로소프트와 파트너의 기술 협력 프로그램 중 하나인 "Project Wandoo"의 Microsoft Sentinel 솔루션 실습 과정입니다. L200을 기준으로, 기본적인 Sentinel의 기능에 대한 HoL을 제공합니다.
